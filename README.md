# Folio

A native macOS ePub reader with AI-powered bilingual translation, rich annotations, and quote-cover generation.

## Features

### Library & Bookshelf
- Browse your imported ePub collection in a responsive cover grid
- Import books via drag-and-drop or toolbar button — metadata is extracted automatically
- Search by title or author in real time
- Filter to recently read books from the sidebar
- Right-click to open, view info, or remove books from your library

### Reading Experience
- Clean, paginated ePub rendering with a macOS Books-style aesthetic
- Three themes: Light, Sepia, and Dark
- Adjustable font size, font family, and line height
- Navigate with arrow keys, click zones, trackpad swipe, or on-screen chevrons
- Table of Contents dropdown for quick chapter navigation
- Reading progress bar with chapter title
- Auto-saves your position — pick up exactly where you left off

### Highlights & Notes
- Select any text to reveal a popup action bar
- Highlight passages in five colors (Yellow, Green, Blue, Pink, Purple)
- Attach text notes to any selection
- View all highlights and notes in the Annotations panel
- Export highlights to Markdown or CSV

### AI Bilingual Translation
- Translate entire books using OpenRouter LLMs
- Read bilingual text with original and translation interleaved paragraph by paragraph
- Translation runs in the background — keep reading while it works
- Pause, resume, or cancel translation at any time
- Toggle bilingual mode on and off
- Export a completed bilingual translation as a new ePub file

### Quote Cover Generator
- Select a passage and generate a beautiful 1080x1080 quote cover image
- Choose from five color themes
- Edit the quote text with live preview
- Includes book cover, title, author, and a subtle watermark
- Save as PNG to your Desktop

### Settings
- Configure your OpenRouter API key (stored securely in macOS Keychain)
- Choose your preferred translation model
- Test your connection before saving

## Requirements

- macOS (Apple Silicon or Intel)
